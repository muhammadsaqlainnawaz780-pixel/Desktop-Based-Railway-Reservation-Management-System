### 📖 Overview
The **Railway Reservation System** is a robust Java-based solution designed to automate train scheduling and ticket booking. It replaces traditional manual entries with a high-performance GUI and real-time database management, ensuring efficiency for both administrators and passengers.

### 🚀 Key Features
* **Authentication:** Secure Admin & User login portal.
* **Management:** Modules for Adding/Updating train schedules.
* **Booking:** Streamlined reservation and seat allocation system.
* **Inquiry:** Real-time search for train availability and schedules.
* **Dashboards:** Clean GUI for viewing passenger and booking details.

### 🛠️ Tech Stack
* **Language:** Java (JDK 23)
* **Frontend:** Java Swing & AWT (Desktop GUI)
* **Backend:** MySQL (JDBC Connectivity)
* **Libraries:** `mysql-connector-j`, `jcalendar-1.4`

### 📁 Core Structure
* `Railway/`: All backend logic and UI source files (`.java`).
* `images/`: Assets and icons for the application.
* `Libraries/`: External JAR dependencies.

### ⚙️ Installation & Setup
1.  **Clone:** Download the repository source code.
2.  **Database:** Import the `.sql` file into your MySQL server.
3.  **IDE:** Open the project in NetBeans or IntelliJ.
4.  **Libraries:** Ensure JAR files are added to the Project Build Path.
5.  **Launch:** Run `Login.java` to start the application.
